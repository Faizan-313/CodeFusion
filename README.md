# 💻 CodeFusion: AI-Driven Online Programming Examination Platform

CodeFusion is a **secure, intelligent, and multi-format web-based platform** designed to revolutionize the way computer science engineering exams are conducted online.  
By integrating **AI-driven proctoring**, a **Smart Teacher Dashboard**, and **multi-format answer support**, it ensures academic integrity, accessibility, and fairness — all within a seamless, real-time environment.

---

## 🧠 Abstract

Traditional online exam systems often fail to ensure integrity or support diverse question formats.  
**CodeFusion** bridges this gap with an **AI-powered, real-time proctoring system** and an **interactive exam workspace** where students can:
- Write **code**
- Draw **diagrams**
- Submit **text responses**
- MCQs also

Instructors receive **alerts** on suspicious activity, enabling **immediate intervention**.  
Built using modern technologies such as **React.js, Node.js, MongoDB**. CodeFusion is modular, scalable, and ready for deployment in universities, bootcamps, and certification platforms.

## 🔒 Key Features

### 👨‍💻 For Students
- 🧩 **Multi-format Answer Interface** — Code, Text, MCQs and Diagram editors in one workspace  
- 🧠 **Lockdown Mode** — Prevents tab switching and external application access  
- 📸 **AI Proctoring** — Webcam-based anomaly detection 

### 👩‍🏫 For Teachers
- 🖥️ **Live Dashboard** — View active students, alerts, and submissions  
- ⚡ **Real-time AI Alerts** — Suspicious activities flagged instantly  
- 🧾 **Exam Management** — Create, schedule, and monitor exams with full control

## 💻 Installation & Setup

### Prerequisites

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/codefusion.git
cd codefusion/backend

# 2. Install dependencies
npm install

# 3. Start the backend
npm run server

# 4. In another terminal, start the frontend
cd ../frontend
npm install
npm start

```
